# sae302

Description de l'application :

Cette application permet de superviser des machines Linux à distance depuis une application Web interactive, vous pourrez visualiser les logs des machines de votre choix à distance de manière centralisée.

Procédure d'installation :

Avoir des machines dans un environnement Linux

Une machine qui jouera le rôle de superviseur, vous installerez le paquet debian associé au projet sur cette machine, les machines à superviser devront être accessibles via ssh depuis le superviseur.
